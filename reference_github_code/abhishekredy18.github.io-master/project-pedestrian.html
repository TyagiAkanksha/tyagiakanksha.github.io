Project Overview

Designed and deployed a real-time pedestrian detection and safety alert system using NVIDIA Jetson Nano for edge AI inference. The system provides intelligent safety alerts for urban traffic scenarios, triggering warnings when pedestrians enter roads during restricted signals or when vehicles approach at unsafe speeds.

System Architecture

  • Detection Models: Implemented and compared YOLO and MobileNet SSD architectures for pedestrian detection
  • Training Data: Models trained on CityPersons and Caltech pedestrian datasets for robust performance in urban environments
  • Edge Deployment: Optimized for NVIDIA Jetson Nano using TensorRT for real-time inference on resource-constrained hardware
  • Streaming Pipeline: Leveraged DeepStream SDK for efficient video processing and multi-stream support

Optimization Techniques

To achieve real-time performance on edge devices, implemented several optimization strategies:

  • Model Quantization: Reduced model precision to INT8 while maintaining detection accuracy
  • Network Pruning: Removed redundant parameters to decrease model size and inference time
  • TensorRT Acceleration: Utilized NVIDIA's TensorRT for optimized inference on Jetson hardware
  • Pipeline Optimization: Efficient frame batching and processing using DeepStream

Safety Alert Features

  • Pedestrian Signal Violation: Triggers audio-visual alerts when pedestrians enter the road during restricted (red) signals
  • Speeding Vehicle Detection: Flashes warning signals for vehicles approaching at unsafe speeds during red lights
  • Real-time Processing: Achieves low-latency detection and alerting for immediate safety intervention
  • Multi-camera Support: Capable of processing multiple camera feeds simultaneously
