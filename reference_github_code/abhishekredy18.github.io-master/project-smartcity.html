Project Overview

Designed a graph neural network (GNN) based system to model relationships across IoT sensors deployed in smart city infrastructure. The system performs spatio-temporal anomaly detection to identify faulty sensors and abnormal patterns in urban environments, improving reliability and operational efficiency.

System Design

The project addresses the challenge of monitoring heterogeneous sensor networks in urban infrastructure:

  • Multi-sensor Integration: Unified data from diverse IoT sensors including traffic monitors, air quality sensors, and energy meters
  • Graph Modeling: Represented sensor network as a graph where nodes are sensors and edges capture spatial and functional relationships
  • Temporal Patterns: Incorporated time-series analysis to capture temporal dependencies in sensor readings
  • Heterogeneous Data: Handled different data types and sampling rates across sensor modalities

GNN Architecture

  • Spatial Component: Graph Convolutional Networks (GCN) to aggregate information from neighboring sensors
  • Temporal Component: Recurrent layers (LSTM/GRU) to model time-dependent patterns
  • Attention Mechanism: Learned importance weights for different sensor relationships
  • Multi-task Learning: Simultaneously predicts sensor values and detects anomalies

Anomaly Detection Capabilities

The system identifies various types of anomalies critical for smart city operations:

  • Sensor Faults: Detects malfunctioning sensors through deviation from expected readings and neighbor patterns
  • Traffic Anomalies: Identifies unusual traffic surges, congestion patterns, and potential incidents
  • Environmental Alerts: Flags abnormal air quality readings and pollution events
  • Energy Irregularities: Detects unusual power consumption patterns indicating faults or inefficiencies

Impact

  • Improved smart city infrastructure reliability through early fault detection
  • Enabled proactive maintenance by identifying failing sensors before complete failure
  • Enhanced incident response by detecting traffic and environmental anomalies in real-time
  • Reduced false alarms through spatio-temporal correlation of sensor data
